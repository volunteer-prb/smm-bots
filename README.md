# smm-bots
Bots for reposting messages in Twitter and TG

# how to install
1. Install npm
```shell script
npm install
```
2. Clone file `.env.example` to `.env` 
```shell script
cp .env.example .env
```
2. Edit file `.env` and fill the variables
```shell script
vi .env
```


# how to run
```shell script
npx ts-node index.ts
```
